# 包结构设计图（目标架构）

```
com.jiuxi/
├── app/                    # 应用层（应用服务、配置等）
│   ├── service/           # 应用服务
│   ├── assembler/         # 数据转换器
│   ├── dto/               # 数据传输对象
│   ├── command/           # 命令对象
│   ├── query/             # 查询对象
│   └── config/            # 应用配置
├── module/                # 领域模块（按上下文划分）
│   ├── auth/
│   │   ├── domain/       # 实体/值对象/领域服务/事件/仓储接口
│   │   ├── app/
│   │   ├── infra/
│   │   └── intf/
│   ├── org/
│   │   ├── domain/
│   │   ├── app/
│   │   ├── infra/
│   │   └── intf/
│   ├── user/
│   │   ├── domain/
│   │   ├── app/
│   │   ├── infra/
│   │   └── intf/
│   ├── role/
│   │   ├── domain/
│   │   ├── app/
│   │   ├── infra/
│   │   └── intf/
│   └── sys/
│       ├── domain/
│       ├── app/
│       ├── infra/
│       └── intf/
├── infra/                 # 基础设施层（持久化、外部服务等）
│   ├── persistence/
│   ├── messaging/
│   ├── cache/
│   ├── external/
│   └── config/
├── intf/                  # 接口适配器层（Web、RPC等）
│   ├── web/
│   ├── rpc/
│   ├── mq/
│   └── scheduler/
├── shared/                # 共享内核（通用工具、公共组件等）
│   ├── common/
│   ├── util/
│   ├── exception/
│   ├── constant/
│   └── security/
└── Bootstrap.java         # 应用启动类
```

说明：模块内统一使用 `intf` 作为适配层包名；历史 `interfaces` 作为过渡名，后续迁移。
